RF4 Backup Tool v1.2.0
======================
Savegame-Backup-Tool fuer Russian Fishing 4
Save Game Backup Tool for Russian Fishing 4

FEATURES / FUNKTIONEN:
  - Backup & Restore Spielstaende / Save files
  - Cloud / NAS Sync (Nextcloud, Samba, USB ...)
  - Automatische Komprimierung / Auto compression

VARIANTEN / VARIANTS:
  - GUI Version:      rf4sa-backup-gui.ps1
  - Terminal Version: rf4sa-backup.ps1
  - Linux/Mac Shell:  rf4sa-backup.sh

QUELLE / SOURCE:

  https://nga.li/rf4git

  Right-click the .ps1 file -> "Run with PowerShell"
  Oder per Startmenue-Verknuepfung / Or via Start Menu shortcut.
